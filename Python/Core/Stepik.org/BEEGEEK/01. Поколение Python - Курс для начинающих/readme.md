# Описание курса:
"Поколение Python: курс для начинающих" знакомит школьников и всех, кому это интересно, с программированием. <br>
Рассказывает об основных типах данных, конструкциях и принципах структурного программирования, используя версию языка Python ветки 3.x. <br>
Выбран Python за ясность кода и быстроту реализации на нем программ. <br>
Цель курса - формирование базовых понятий структурного программирования. <br>
В нем 8 модулей с теоретическими и практическими материалами и заданиями. <br>

Решения проверяет автоматическая система, поэтому обратную связь вы получите быстро. Если у вас возникнут вопросы, команда курса даст советы и подсказки. Кроме того, проблемы можно обсуждать с однокурсниками в комментариях к задачам. <br>
Курс – победитель конкурса Stepik Awards 2020 в номинации "Лучший бесплатный онлайн-курс размещенный на платформе Stepik". <br>
Сертификат о прохождении этого курса позволяет участвовать в конкурсе «Талант 20.35» <br>
Этот курс –  первый из серии “Поколение Python”, в которую также входят Курс для продвинутых и Курс для профессионалов. <br>
Первые два курса серии открыты бесплатно. Поблагодарить и поддержать организаторов можно на странице: https://iqcharity.ru/kak-pomoch. <br>

Ваша поддержка поможет нам и дальше делать IT-образование доступнее! <br>
